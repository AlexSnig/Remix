# Claude Instructions

Use `AGENTS.md` as the canonical project instruction file.

Before editing code:

1. Read `AGENTS.md`.
2. Read `PROJECT_MEMORY.md`.
3. Inspect the files directly related to the requested change.
4. Run `npm run lint` and `npm run build` when dependencies are installed.

Do not treat this as a generic React app. It is a browser-based museum kiosk sensor with camera, audio, fullscreen, wake-lock, IndexedDB, and mobile constraints.
